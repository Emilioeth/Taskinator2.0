  ## Description
  Taskinator 2.0 is an updated version to Taskinator, with an improved user interaction its now easier to create, move and delete tasks as you wish, now you can drag tasks to rearrange them, or to delete them, also better experience for mobile users.

  [Link](https://emilioeth.github.io/Taskinator2.0/)
